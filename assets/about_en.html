About this software

This software show old map in Tokyo, with Google Maps JavaScript API V3.

This software accesses following services.

  • Historical Agro-Environment Web Map Service (Digitized by Historical Agro-Environemnt Browsing System)
  • Municipality Name and Cetroid List Service (Original Data: Digital National Land Information (Administrative Area))

About the old maps

Tokyo 1:5000
1:5000 map in Tokyo Central. Created on 1883 (Meiji 16).
Rapid Survey Map in Kanto Plain
1:25000 map in Kanto Plain. Created on 1880 (Meiji 13) - 1886 (Meiji 19).
