このソフトウェアについて

このソフトウェアは、Google Maps JavaScript API V3 を用いて、東京の古い地図を表示するものです。

このソフトウェアは次のサービスにアクセスします。

  • 歴史的農業環境WMS配信サービス (データ作成: 歴史的農業環境閲覧システム)
  • 市区町村名・重心一覧検索サービス (国土数値情報(行政区域データ)を使用しています)

古い地図について

東京図測量原図
1883(明治16)年に作成されました。東京中心部の地図で、縮尺は5000分の1です。
関東平野迅速測図
1880(明治13)-1886(明治19)に作成されました。関東平野の地図で、縮尺は2万分の1です。
